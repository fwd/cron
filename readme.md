> A NodeJS wrapper around setInterval for cron like repetitive tasks.

## Install

```sh
npm install @fwd/cron
# or
yarn add @fwd/cron
```

## Usage

```js

const cron = require('@fwd/cron')

cron(() => {
	console.log("Hello, World")
}, 'every 2 seconds')

```
